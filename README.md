# Gpu-eda Tap

## How do I install these formulae?

`brew install gpu-eda/tap/<formula>`

Or `brew tap gpu-eda/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "gpu-eda/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
